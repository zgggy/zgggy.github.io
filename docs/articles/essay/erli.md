# 而立

> 咬文嚼字而已

@ 2026-4-30

& 散文

“而立”，是个很奇怪的词。奇怪之处全在那个“而”字。一个毫无实义的连词，凭空插在“立”字前面，像脸上长了一块多余的骨头，硌得让人难受。可能若只有一个“立”字，就很难再去组一些朗朗上口的四字词语，但这个多余的过于明显的前缀，让三十岁的站立，仿佛本身就带上了某种勉强和迟疑。

这个词让我别扭了许多年，它像一种集体默认的妥协：所有人都明白一件事情虽未完成，但接受它不完整的样子。像所有人知道一件事本应该如何，但接受它未如愿的样子。它就这样装下我所有未完成的事，和所有未说出口的话。

不想接受它，不想脱口而出这个词。我甚至觉得说出这个词是违反了这门语言应该持有的逻辑，是丧失了对这门语言的基本尊重。我现在站在水里，看着对岸的土地，看着那个多余的“而”字，被每个人举在头顶，纹在背上，我不敢抬脚。

是不是接受了“而”，才算“立”呢？

我想，我宁愿接受卡关。

我站在水里泡脚算了。卡关也许不是坏事，永远停在二十九岁，永远到不了“不惑”。不惑，那就一直惑。在水里泡久了，就往下扎根，变成一棵水中的树好了，就变成一棵树吧。
