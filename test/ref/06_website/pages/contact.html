常见问题

快速找到您需要的答案

❓

如何开始使用 Starletto？

请访问我们的 GitHub 仓库，阅读快速开始文档。 我们提供了详细的安装指南和示例代码，帮助您快速上手。

❓

Starletto 是免费的吗？

是的，Starletto 是完全开源免费的项目。 我们采用 Apache 2.0 许可证，您可以自由使用、修改和分发。

❓

如何贡献代码？

请查看 GitHub 上的贡献指南。我们欢迎所有形式的贡献， 包括代码、文档、测试等。提交 Pull Request 前请确保通过所有测试。

❓

支持哪些平台？

目前主要支持 Linux 平台（Ubuntu 20.04+）。 我们计划在未来支持更多平台，包括 Windows 和 macOS。

❓

有商业支持吗？

我们提供商业技术支持服务，包括定制开发、技术咨询、培训等。 请通过 business@starletto.io 联系我们了解详情。

❓

如何报告 Bug？

请在 GitHub Issues 中提交 Bug 报告。 提供详细的复现步骤、系统环境和日志信息，帮助我们快速定位问题。
