关于我们

致力于让自动驾驶开发更简单、更高效

我们的愿景

Starletto 诞生于对自动驾驶技术的热爱与追求。我们相信，自动驾驶技术 将深刻改变人类的出行方式，而优秀的开发框架是推动这一变革的重要基石。

我们的愿景是为自动驾驶开发者提供一个高性能、易扩展、可维护的系统框架， 让更多人能够参与到自动驾驶技术的创新中来，共同推动行业发展。

我们汲取了 Apollo、Autoware 等优秀开源项目的设计理念，结合实际工程经验， 打造出更适合现代自动驾驶开发的系统架构。

技术理念

指导 Starletto 设计与开发的核心原则

🎯

性能至上

自动驾驶系统对实时性要求极高。我们从架构设计到代码实现， 都将性能作为首要考虑因素，确保系统能够满足实时性要求。

🔧

模块化设计

清晰的模块划分与接口定义，让开发者能够灵活组合与扩展。 每个模块都可以独立开发、测试和替换，降低系统复杂度。

📖

文档驱动

完善的文档是项目成功的关键。我们提供详尽的开发文档、 API 参考和最佳实践指南，帮助开发者快速上手。

🤝

开放协作

我们拥抱开源精神，欢迎社区贡献。通过开放的协作模式， 汇聚全球开发者的智慧，共同推动项目发展。

🔒

安全可靠

安全性是自动驾驶系统的生命线。我们在设计中充分考虑安全性， 提供多重保护机制，确保系统稳定可靠运行。

持续创新

自动驾驶技术日新月异。我们保持对新技术的敏感度， 持续优化架构设计，引入最新技术成果，保持项目活力。
