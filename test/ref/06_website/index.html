STARLETTO

一个高性能、模块化的自动驾驶系统框架，提供完整的感知、规划、控制解决方案。 借鉴 Apollo 和 Autoware 的最佳实践，为开发者打造更简洁、更高效的开发体验。

starletto 采用现代 C++ 架构，支持多传感器融合、高精度定位、智能路径规划和实时车辆控制， 为自动驾驶技术的研发和应用提供强大的基础平台。
