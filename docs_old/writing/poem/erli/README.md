# 而立

而立，是个很怪的词。
 
很怪，真的很怪。
 
“而”是个连词，却被硬塞在“立”前面，凑成了三十岁的代称。
 
明明一个“立”字就够了。
 
我别扭了好几年。
 
我觉得它像一根刺。
 
像我人生里所有的遗憾，像一些醒不来的梦魇。
 
它就不该在那里。
 
它就不该在那里。
 
哎，可是它就在那里。
 
千年来，每个人都在说而立，而立，而立之年。
 
每个人都接受了这个多余的字。
 
那咋办呀？
 
那我接受吗？
 
我该接受吗？
 
我能接受的了吗？
 
我不确定。
 
我好像并不接受。
 
但看起来如果我不接受它，
 
我就会永远停在而立的前一刻，
 
永远走不到不惑。
 
卡关了。
 
卡关了也挺好。
 
永远29岁。
 
不能不惑，就一直惑。
 
不往前走，就往脚下扎根。
 
变成一棵树好了。
 
就变成一棵树吧。
